# terraform-azure-reference
This is a reference project to setup a reference infrastructure environment on azure with terraform




## Install terraform


```bash
  brew install terraform
```

## Create a dev.tfvars file

```bash
client_id=""                                                                                                                                                       
client_secret=""                                                                                                                                               
tenant_id=""                                                                                                                                                       
subscription_id=""                                                                                                                                                 
prefix="dev"
location="West Europe"

```
